# alpha-shop-front-end-vue

## 專案介紹
使用 Vue 2.0 製作的簡易購物訂單頁面

## 功能
1. 使用者按上、下一步能夠切換結帳區狀態
2. 使用者按上、下一步能夠切換表單畫面
3. 使用者能在購物車調整商品數量，價格能即時反應到「小計」
4. 使用者可以依需求選擇運送方式
